# Phase 2 Plan (Draft)

## Major Features Planned

### 1. Improved Intelligence Layer

We plan to expand the learning assistant’s intelligence capabilities by implementing:

* Hybrid search (semantic + keyword retrieval)
* Personalized hint progression based on student behavior
* Smarter retrieval ranking for more accurate curriculum guidance
* Better follow-up questioning to encourage deeper learning instead of answer copying

This feature will strengthen the assistant’s ability to guide students while maintaining the project’s core “learn without direct answers” philosophy.

### 2. Advanced Instructor & Admin Analytics Dashboard

We plan to build a more advanced analytics and monitoring system for instructors and administrators, including:

* At-risk student indicators
* Struggle heatmaps by topic or assignment
* Common misconception reports
* Curriculum usage analytics
* Instant curriculum update controls

This feature will provide actionable insights to instructors and administrators so they can better support students and continuously improve the curriculum.

---

## Improvement Area Targeted

### Primary Focus: User Experience (UX)

Phase 2 will heavily improve the usability and overall experience of the platform by:

* Making the assistant feel more interactive and personalized
* Improving dashboard clarity and navigation
* Adding mobile responsiveness
* Creating smoother student and instructor workflows

### Secondary Focus: Evaluation & Performance

We will also focus on:

* Improving retrieval accuracy and response quality
* Measuring effectiveness of the guardrail system
* Optimizing backend response times and search performance
* Enhancing logging and analytics reliability
